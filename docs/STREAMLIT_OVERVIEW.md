## 🎯 What It Does

The console provides **real-world API testing** for your multi-tenant RAG system:

```
┌─────────────────────────────────────────────────────────────┐
│              GraphMind Testing Console (Streamlit)           │
├──────────────────┬──────────────────────────────────────────┤
│ SIDEBAR          │ MAIN TABS                                │
├──────────────────┼──────────────────────────────────────────┤
│ 🔐 Auth          │ 🤖 Agents                                │
│ ⚙️  Settings     │ 📚 Knowledge Base                        │
│                  │ 🔍 RAG Query (MAIN)                      │
│                  │ 📈 Analytics                             │
│                  │ 💳 Billing (optional)                    │
│                  │ 🔧 Debug                                 │
└──────────────────┴──────────────────────────────────────────┘
```

### Complete Flow:
```
1. SIGNUP → Register new user (JWT token stored)
2. LOGIN → Authenticate (token in Authorization header)
3. AGENTS → Create agent, select active agent
4. KB → Create knowledge base, upload text
5. QUERY → Ask question, view answer + sources + stats
6. ANALYTICS → Track latency, history, metrics
7. BILLING → See cost (if enabled)
```

## ✨ Key Features

### ✅ Authentication
- Signup with email/password/name
- Login flow with JWT token
- Token stored in Streamlit session
- Automatic Authorization header on all requests
- Logout support

### ✅ Agent Management
- Create agents with name, description, system prompt
- List all agents (dropdown)
- Select active agent for KB/Query operations
- Agent-specific KB isolation

### ✅ Knowledge Base
- Create multiple KBs per agent
- Upload text to KBs (no file size limit in this version)
- Track KB metadata

### ✅ RAG Query (Core Feature)
- Input: User question
- Output:
  - **Answer**: Generated by LLM
  - **Sources**: Expandable section showing retrieved chunks
  - **Stats**: Full metrics (tokens, latency, LLM source, cost if billing enabled)
- Latency tracking for performance monitoring

### ✅ Analytics
- **Total Queries**: Count of all queries made
- **Avg Latency**: Performance metric (goal: < 2000ms)
- **Query History**: Last 10 queries with latencies
- All tracked in session state

### ✅ Billing (Conditional)
- Only appears if backend has `ENABLE_BILLING=True`
- Shows:
  - Tenant cost (your total spend)
  - API call count
  - Total tokens used
- Safe to enable: no cost during testing at local rates

### ✅ Error Handling
- Clear error messages
- HTTP status codes displayed
- API response inspection in Debug tab
- Timeouts handled gracefully

### ✅ Developer Experience
- **Debug Tab**: Inspect token, agent ID, API URL, session state
- **Fast Interactions**: All UI responsive, no lag
- **Minimal UI**: Focus on capability, not design
- **Real-World Flow**: Tests actual API endpoint usage

## 🚨 Common Issues & Fixes

### Issue: "API connection failed"
**Cause**: Backend not running
**Fix**: `uvicorn app.main:app --reload` (Terminal 1)

### Issue: "Unauthorized" (401)
**Cause**: Token expired or invalid
**Fix**: Logout and login again

### Issue: Query takes > 5 seconds
**Cause**: Slow embedding or LLM API
**Fix**: Check DeepInfra API status, check Neo4j performance

### Issue: "Agent not found"
**Cause**: Agent deleted or wrong ID
**Fix**: Create new agent, select it again

### Issue: Upload fails
**Cause**: Invalid text or KB not found
**Fix**: Ensure KB exists, paste valid text, try smaller chunk

**See `STREAMLIT_QUICK_REFERENCE.md` for more troubleshooting**
